Approach

Predicting multiple tags for specific math questions is a multi-label classification problem

  • Parse each question file to extract the keywords and question text
  • Parse each question's text to extract embedded LaTeX expressions
  • Tokenize question text and LaTeX expressions
  • Train classifiers using One-vs-the-rest (OvR) multiclass/multilabel strategy
    • Also known as the binary relevance method, consists of fitting one classifier per class
  • Naive Bayes and SVM with linear kernel
